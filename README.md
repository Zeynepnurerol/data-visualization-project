# Dünya Mutluluk Analizi

Bu projede Python kullanılarak ülkelerin mutluluk skorları analiz edilmiştir. Veri seti kullanılarak ülkelerin mutluluk seviyeleri incelenmiş ve grafiklerle görselleştirilmiştir.

## Kullanılan Teknolojiler
- Python
- Pandas
- Seaborn
- Matplotlib

## Projenin Amacı
Bu proje, ülkelerin mutluluk skorlarını analiz ederek veri görselleştirme tekniklerini uygulamayı amaçlamaktadır.

## Yapılan Analiz
- Veri seti CSV dosyasından okunmuştur
- Ülkelerin mutluluk skorları incelenmiştir
- İlk 7 ülke için çizgi grafiği oluşturulmuştur
- Grafik üzerinde ülkeler arasındaki mutluluk farkları analiz edilmiştir

## Grafik Yorumu
- Switzerland en yüksek mutluluk skoruna sahip ülkedir.
- Holland en düşük mutluluk skoruna sahip ülkedir.
- İlk üç ülke mutluluk skorlarında öne çıkmaktadır.

## Veri Seti
Proje, Dünya Mutluluk Raporu veri seti kullanılarak hazırlanmıştır.
